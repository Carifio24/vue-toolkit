Cosmic Data Stories Vue Toolkit

Getting Started

As the name suggests, the Vue Toolkit is built using Vue, an open-source JavaScript framework. The toolkit leverages the power and versatility of Vue to create a number of flexible components, together with some utility functions to make common operations easier. Before you can get started building, though, you'll (possibly) need to install a few things.

Node.js

The first thing that you'll need to use the toolkit is Node.js, an open-source, cross-platform JavaScript runtime environment. To get started, simply download Node.js from the linked page and install it on your computer.

Yarn

Technically, this one is optional - you can write Vue-based applications using Node's built-in package manager NPM without any issue. However, we on the CosmicDS team use yarn for our package management, and that's what we'll use in our examples on this webpage. Yarn uses the same registry of packages as npm, so everything you find on NPM is fair game. (If you want to use NPM, that's fine - you just may need to translate some commands here and there).

Install the toolkit

Once you have Node.js and a Vue project set up, you can install the toolkit by simply running

yarn add @cosmicds/vue-toolkit

If the part about setting up a Vue project is totally new to you, don't worry! You can follow our guide for creating a story - we'll help your story get up and running in no time.
