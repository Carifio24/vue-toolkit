Cosmic Data Stories Vue Toolkit

What Is It?

The Cosmic Data Stories Vue Toolkit is a package of versatile, flexible components designed to help simplify the process of making educational web-based interactives. The the name suggests, this toolkit is built using Vue, an open-source JavaScript framework. The toolkit leverages the power and versatility of Vue to create a number of reusable components that can be used in a variety of data stories. The toolkit also includes some utility functions to help make common operations easier.

Getting Started with the Toolkit

While we've aimed to make the toolkit as simple to use as possible, there are a few things that you'll need to install, if you don't have them already. (Note that users of your interactives won't need to do any of this - this step is necessary for developers only).

The first thing that you'll need to use the toolkit is Node.js, an open-source, cross-platform JavaScript runtime environment. To get started, simply download Node.js from the linked page and install it on your computer.

Technically, this one is optional - you can write Vue-based applications using Node's built-in package manager NPM without any issue. However, we on the CosmicDS team use yarn for our package management, and that's what we'll use in our examples on this webpage. Yarn uses the same registry of packages as npm, so everything you find on NPM is fair game. (If you want to use NPM, that's fine - you just may need to translate some commands here and there).

Set up a Vue project

Once you have the necessary tools installed, you'll want to set up a Vue project to start building your story with the toolkit.

Install the toolkit

Once you have Node.js and a Vue project set up, you can install the toolkit by simply running

yarn add @cosmicds/vue-toolkit
